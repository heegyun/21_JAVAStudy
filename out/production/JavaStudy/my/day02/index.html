• 클래스의 구조
    1) 패키지 선언: 최상단에 위치해야 한다. import 문 보다도 먼저 와야 함.
    2) import 문: 사용하고자 하는 패키지 경로를 적는다.
    3) class 선언: class키워드로 선언하고 클래스 이름을 기재 이 때 주의. 클래스명 == 파일명
    4) 클래스의 구성원(Member)
    1> 속성(멤버변수): ex) int a=10; if(a==10)
    종류: ㄱ)클래스 변수: static변수
    ㄴ)멤버변수: instance변수
    2> 생성자(Constructor) :
    -멤버변수의 초기화
    -객체를 생성할 때 호출한다.
    -생성자 이름 == 클래스 이름
    -반환 타입을 명시하지 않는다.
    int a;
    ex) public Demo(){
    a=100;//멤버 변수의 초기화
    }
    3> 메소드(Method)
    종류: ㄱ)클래스 메소드: static 메소드
    ㄴ)멤버 메소드: non-static 메소드
